# Expense Tracker

A full-stack Expense Tracker application built using the MERN Stack that enables users to manage their personal finances efficiently. The platform provides secure authentication, expense and income tracking, interactive analytics, profile management, and downloadable PDF reports.

## Features

### Authentication & Security

* User Registration and Login
* JWT-based Authentication
* Protected Routes and Authorization
* Secure Password Management

### Expense Management

* Add Income and Expense Transactions
* Edit and Delete Transactions
* View Complete Transaction History
* Real-Time Balance Calculation

### Analytics & Visualization

* Interactive Charts for Financial Insights
* Income vs Expense Analysis
* Expense Distribution Analytics
* Financial Summary Dashboard

### Reports

* Generate and Download Financial Reports in PDF Format
* Detailed Transaction Records

### User Profile Management

* Update Profile Information
* Change Name
* Update Email Address
* Change Password Securely

### User Experience

* Responsive Design
* Clean and Modern UI using Vanilla CSS
* Fast and Intuitive Navigation

## Tech Stack

### Frontend

* React.js
* Vanilla CSS
* Axios
* Chart.js / Recharts

### Backend

* Node.js
* Express.js
* JWT Authentication
* Bcrypt.js

### Database

* MongoDB
* Mongoose
